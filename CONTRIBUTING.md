# Contributing

Thank you for your interest in this repository.

This is a personal portfolio website for Vahhab Piranfar. Contributions are limited to corrections, technical improvements, broken-link fixes, accessibility improvements, and suggestions related to the website structure.

## Acceptable contributions

- Fixing typographical errors
- Reporting broken links
- Improving accessibility or responsiveness
- Suggesting improvements to project pages
- Reporting technical issues with the website

## Not acceptable

- Reusing personal images, CV content, biography, academic descriptions, or project descriptions without permission
- Submitting unrelated promotional material
- Making changes that misrepresent the author, projects, publications, or affiliations

## How to contribute

Please open an issue first to describe the proposed change. For small corrections, a pull request may be submitted directly.

All contributions should be respectful, accurate, and relevant to the purpose of this repository.
