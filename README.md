# Vahhab Piranfar Personal Site

## Project Description
This is the source for my personal research website, built with Quarto and published with GitHub Pages at the root `piranfar.github.io` address.

## Features
- **Portfolio**: A collection of my work and projects.
- **About Me**: A brief introduction to my background and skills.
- **Contact**: Options for reaching out to me.

## Live Site
You can view the live version of the site at [https://piranfar.github.io](https://piranfar.github.io).

## Deployment
Pushes to the `main` branch render the Quarto site and publish the generated `docs` directory through GitHub Pages.
